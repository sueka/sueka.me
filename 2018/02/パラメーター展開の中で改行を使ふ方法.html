パラメーター展開の中で改行を使ふ方法

投稿日
2018年2月8日

改行を変数にするだけです。

実例

#!/bin/sh
#
# Teletype for Atom がインストールされてゐるかどうかを返すスクリプト
#

set -eu

LF='
'
PACKAGES=$(apm list -bip | sed -n 's/@.*//p')

if [ -n "${PACKAGES##*${LF}teletype${LF}*}" ]; then
  exit 1
fi
